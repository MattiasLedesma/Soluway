# Soluway — Sitio Web Institucional

Sitio estático de [Soluway](http://soluway.io), empresa de AI y automatización para pymes y negocios locales.

## Stack

- HTML + CSS (Tailwind CDN)
- Font Awesome (iconos)
- Google Fonts (Inter)
- Formulario de contacto via [Formspree](https://formspree.io)

## Estructura

```
├── index.html       # Sitio completo (single-page)
├── assets/
│   └── logo.png     # Logo Soluway
└── CNAME            # Dominio personalizado para GitHub Pages
```

## Secciones

- Hero
- Quiénes somos
- Posibilidades (casos de uso para pymes)
- Proyectos
- Resultados / métricas
- Reseñas
- Contacto (formulario → soluway.group@allaria.com)

## Deploy

El sitio está publicado con **GitHub Pages** en:
- https://mattiasledesma.github.io/Soluway/
- http://soluway.io (dominio personalizado)

Para actualizar: hacer push a `main` y GitHub Pages publica automáticamente en ~1 minuto.
